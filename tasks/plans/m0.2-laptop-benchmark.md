# M0.2: Laptop Performance Benchmark — Ingest, Analyze, Report

Benchmark laptop performance under heavy multi-tool developer workloads. Collect raw data locally, get it into the artifact store, then let the agent workforce analyze and produce deliverables autonomously.

## Workload Profile

Combinations of: multiple Claude Code instances, browser tabs, Docker Desktop with Node containers, VS Code windows. Measure what happens as load stacks up.

## Pipeline

1. **Data collection** (manual, local) — capture performance metrics across scenarios, save as files in repo
2. **Ingest** — Pi agent in RPC mode reads local files, uploads into artifact store
3. **Agent workforce** — Paperclip issue triggers autonomous analysis and report generation
4. **Deliverables** — detailed performance report + social media carousel

## Success Criteria

- [ ] Raw benchmark data collected and ingested into artifact store
- [ ] Agent workforce completes analysis and report without manual intervention beyond initial issue
- [ ] Report contains actionable findings about bottlenecks and resource limits
- [ ] Carousel content is ready for posting on social media
- [ ] Full pipeline visible in OpenObserve traces

## Dependencies

- Artifact store operational
- Agent workforce functional (M0.1)

## What This Tests

How well the agent workforce handles a real analytical task end-to-end when given data and a goal, without hand-holding on method.
